# Clinic Management System - نظام إدارة العيادة

تطبيق ويب شامل لإدارة عيادة طب الأسنان، يوفر إدارة متكاملة للمرضى والمواعيد والعلاجات.

## المميزات

### إدارة المرضى
- إضافة وتعديل وحذف بيانات المرضى
- البحث السريع بالاسم أو رقم الهاتف
- الملف الطبي الكامل مع:
  - المعلومات الشخصية
  - فصيلة الدم والحساسية
  - التاريخ المرضي
  - ملاحظات الطبيب

### إدارة المواعيد
- عرض المواعيد (قائمة أو تقويم)
- إضافة مواعيد جديدة مع اختيار المريض والطبيب والعلاج
- تحديد حالة الموعد (مجدول/مكتمل)
- تصفية حسب الحالة

### الخطط العلاجية
- إنشاء خطط علاجية متكاملة
- تتبع التكلفة الإجمالية والمدفوع والمتبقي
- متابعة مراحل العلاج

### المؤشرات والإحصائيات
- إيرادات اليوم والشهر
- عدد المرضى والعلاجات
- إنتاجية الأطباء
- رسوم بيانية للعلاجات الأكثر طلباً

### الميزات الإضافية
- تصدير البيانات (JSON/CSV)
- استيراد البيانات
- واجهة مستخدم سهلة وعصرية
- دعم العربية

## كيفية التشغيل

### المتطلبات
- Node.js 18+

### التثبيت والتشغيل

```bash
# تثبيت المتطلبات
npm install

# تشغيل خادم التطوير
npm run dev

# بناء للإنتاج
npm run build
```

### المسارات المتاحة
- `/dashboard` - لوحة التحكم الرئيسية
- `/patients` - قائمة المرضى
- `/patients/:id` - ملف المريض
- `/appointments` - المواعيد
- `/indicators` - المؤشرات والإحصائيات
- `/settings` - الإعدادات والنسخ الاحتياطي

## الأمان

- التحقق من صحة جميع المدخلات
- تنظيف المدخلات (Sanitization) لمنع XSS
- استخدام معرفات فريدة آمنة (UUID)
- التحقق من أرقام الهواتف والبريد الإلكتروني
